Barista VR  🍌🐒

Project Description 🍌
Barista VR is an early alpha XR/VR project built in Godot using OpenXR.  
The idea is to simulate a small café environment where the player can interact with objects and explore the space in virtual reality. 🍌🐒  

The main goal is to demonstrate basic VR interaction mechanics and environment setup.

---

Idea and Goal 🍌
The goal of this project is to:
- Create a simple VR café environment 🍌
- Allow player interaction with objects 🍌🐒
- Demonstrate basic VR mechanics using OpenXR 🍌
- Build a foundation for a future barista simulator 🍌🐒  

---

Steps Completed 🍌🐒

- OpenXR setup and VR configuration 🍌  
- Basic café environment created 🍌  
- Added furniture (tables, chairs, shelf, etc.) 🍌🐒  
- Added textures and materials 🍌  
- Imported 3D models (GLB assets) 🍌🐒  
- Implemented basic interaction system (pickable objects) 🍌  
- Camera and player setup 🍌🐒
